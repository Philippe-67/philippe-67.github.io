Définition

Une image Docker est un modèle léger, autonome et exécutable qui inclut tout ce qu'il faut pour exécuter une application : code, runtime, bibliothèques et paramètres système par défaut.

Création d'Images Docker

  • Utilisez un fichier Dockerfile pour spécifier les instructions nécessaires à la création de l'image.
  • Exécutez la commande docker build pour construire une image à partir d'un Dockerfile.
  • Taguez vos images avec docker tag pour les identifier facilement.

Utilisation des Images Docker

  • Déployez des conteneurs avec docker run en utilisant une image Docker.
  • Partagez vos images via Docker Hub ou d'autres registres Docker.
  • Utilisez docker pull pour récupérer des images depuis un registre.

Exemples Pratiques

Exemple de Dockerfile :

            
              {`FROM node:14
WORKDIR /app
COPY package.json ./
RUN npm install
COPY . .
CMD ["node", "app.js"]`}

Commande pour créer une image :

            docker build -t mon-image .

Commande pour exécuter un conteneur :

            docker run -p 3000:3000 mon-image
